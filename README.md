# Airbnb Price Prediction Project

End-to-end machine learning project for predicting Airbnb listing prices.

## Project Structure
- Data loading and validation
- Data auditing and cleaning
- Feature engineering
- Scikit-learn pipeline
- Model training & evaluation
- Baseline comparison
- Error analysis

## Model
Linear regression baseline with preprocessing pipeline.

## Future Improvements
- Tree-based models (XGBoost, LightGBM)
- Hyperparameter tuning
- Feature importance analysis
- Model deployment
